## üìö About

Nokuvela Academy is a newly established education centre supporting neighbouring schools in KwaMusi Area, Nokuvela Site, Nongoma, Zululand. We provide free tutelage, mentorship, and skills training from local graduates and artisans, empowering youth and community members to become self-sufficient individuals.

### Mission
We create a space where learners can access quality education, cultivating creative and logical faculties through comprehensive curriculum including language foundations, mathematics, physical sciences, computer literacy, music, art, and philosophy.

### Vision
We envision a university-like institution with proper accreditation, attracting learning enthusiasts from neighbouring communities and building a decentralized education system that proves quality education is accessible to all.

## üöÄ Getting Started

### Prerequisites

No special prerequisites needed. Just a modern web browser.

### Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/nokuvela-academy.git
```

2. Navigate to the project directory
```bash
cd nokuvela-academy
```

3. Open `index.html` in your web browser
```bash
# On macOS
open index.html

# On Linux
xdg-open index.html

# On Windows
start index.html
```

Or simply double-click the `index.html` file.
